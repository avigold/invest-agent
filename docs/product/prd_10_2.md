# PRD 10.2 — Benchmark Comparison (Feature Roadmap #4)

## Problem

Users cannot compare a stock's performance against its country equity index. Bloomberg and YCharts show this comparison natively — it is essential for evaluating whether a stock is outperforming or underperforming the market.

## Solution

Add a toggle to the stock chart that overlays the country's equity index (e.g. S&P 500 for US stocks, AEX for Dutch stocks). When enabled, both lines are normalised to percentage change from the period start, starting at 0% and diverging based on relative performance.

## Scope

Country index only. Sector ETFs are US-only and deferred to a future iteration.

## Data Sources

- **Country index mapping**: `config/investable_countries_v1.json` → `equity_index_symbol`, `equity_index_name`
- **Index prices**: `CountrySeries`/`CountrySeriesPoint` tables (already ingested daily via `app/ingest/marketdata.py`)
- **Stock prices**: `CompanyPriceHistory` JSONB (existing)

## API Change

Extend `GET /v1/company/{ticker}/chart`:
- Add optional `benchmark=index` query param
- When present, response includes additional `benchmark` field:
  ```json
  {
    "benchmark": {
      "name": "S&P 500",
      "symbol": "^GSPC",
      "country_iso2": "US",
      "points": [{"date": "2025-03-24", "value": 0.0}, ...],
      "stock_normalised": [{"date": "2025-03-24", "value": 0.0}, ...]
    }
  }
  ```
- Both `points` and `stock_normalised` contain percentage change values from the first overlapping date
- Without `benchmark` param, response is unchanged (backward compatible)

## UI

- Toggle button right-aligned in the period selector row: "vs S&P 500" / "vs Index"
- When active:
  - Chart switches to normalised % mode (both series start at 0%)
  - Stock: blue AreaSeries, Benchmark: amber LineSeries
  - Volume bars hidden (meaningless for % comparison)
  - Legend below PriceHeader shows both series with crosshair values
  - PriceHeader still shows absolute price, crosshair tooltip shows %
- When inactive: chart returns to absolute price view with volume bars

## Files Changed

| File | Change |
|------|--------|
| `app/api/routes_companies.py` | Add `benchmark` query param, query country index prices, normalise, include in response |
| `web/src/lib/queries.ts` | Update query key and hook to accept `benchmark` param |
| `web/src/components/StockChart.tsx` | Add LineSeries overlay, toggle button, legend, normalised % mode |

## Acceptance Criteria

1. Toggle appears on all stock charts next to period selector
2. US stocks show "vs S&P 500", NL stocks show "vs AEX", etc.
3. Both lines start at 0% and diverge based on performance
4. Crosshair shows both values in legend
5. Switching periods while benchmark is on refetches correctly
6. Toggle off restores absolute price view with volume bars
7. Companies without index data: toggle does nothing gracefully
8. `npm run build` succeeds
9. Backward compatible: chart API without `benchmark` param unchanged
